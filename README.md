## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3131
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate

# pm2运行生产模式
$ npm run pm2
```
